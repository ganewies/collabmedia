## 1. How2Setup this
### I. Requirements

- NodeJS 20-24 (the recommended version is **22**) (for Linux, **WE DON'T RECOMMEND YOU INSTALLING NODEJS FROM APT IF YOU ARE ON DEBIAN OR DEBIAN BASE DISTROS**)
- Yarn (to install Yarn: `npm i -g yarn corepack`)

### II. Install & Run

When you've followed every requirements, you can get past this step.

To install necessary packages, do `yarn`; You will need to build the project by doing `yarn build`

When done, just run it by simply doing `yarn serve`!
